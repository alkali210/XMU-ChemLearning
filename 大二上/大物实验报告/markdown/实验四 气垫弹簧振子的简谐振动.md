# 实验四 气垫弹簧振子的简谐振动



## 一、实验目的

  1. 学会气垫装置的调整与实验方法。
  2. 考察弹簧振子的振动周期与振动系统参量的关系。
  3. 学习用图解法求等效弹簧的倔强系数和有效质量。

## 二、实验仪器

  1. **仪器用具**
气垫导轨、滑块 （包括挡光片）、光电门、智能数字测时器、弹簧。
  2. **仪器描述**
具体介绍见附录 “气垫实验基本知识”。

## 三、实验原理

  1. **弹簧的倔强系数**

弹簧在机械装置中占有重要的地位，弹簧的倔强系数表征弹簧的重要特性。在一定的外力下弹簧的形变、弹簧做周期性振动的频率均与倔强系数有关。

根据胡克定律，在弹性限度内，弹簧的伸长量 x 与它所受的拉力成正比

$$F = - k x ,$$

此比例系数 k 就是弹簧的倔强系数

$$k = - \frac{F}{x} \quad (\text{N/m}) 。$$

  2. **弹簧振子的简谐运动方程**

两个倔强系数分别为 $k_1$ 和 $k_2$ 的弹簧系住一个质量为 $m_1$ 的滑块，两弹簧的另外两端分别固定在导轨的端面上。滑块在导轨上作直线往复振动。当滑块处于平衡位置时，两弹簧的伸长量分别是 $x_{01}$ 和 $x_{02}$，满足 $-k_1 x_{01} + k_2 x_{02} = 0$。若略去阻尼影响，当 $m_1$ 离平衡点 O 为 x 时，$m_1$ 所受的弹性回复力为 $[ -k_1 ( x + x_{01} ) ]$ 和 $[ -k_2 ( x - x_{02} ) ]$。根据牛顿第二定律，滑块的运动方程为

$$-k_1\left(x+x_{01}\right)-k_2\left(x-x_{02}\right)=m_1\frac{\mathrm{d}^2 x}{\mathrm{d} t^2},$$

$$-\left(k_1+k_2\right) x=m\frac{\mathrm{d}^2 x}{\mathrm{d} t^2},$$

式中，$m = m_1 + m_0$ ，称为振动系统的有效质量，其中的 $m_0$ 是弹簧的有效质量，$m_1$ 是滑块质量。令 $k = k_1 + k_2$，则

$$-k x=m\frac{\mathrm{d}^2 x}{\mathrm{d} t^2},$$

此方程的解为

$$x = A\sin\left(\omega t+\psi_0\right) 。$$

这说明滑块是在作简谐振动。其中

$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{k_1 +k_2}{m}}$$

是振动系统的圆频率，由振动系统决定；A 是振幅，$\psi_0$ 是初位相，由起始条件决定。系统的振动周期

$$T = \frac{2\pi}{\omega} = 2\pi \sqrt{\frac{m}{k}} 。$$

当 $m_1 \ll m_0$ 时，$m_0 = \frac{m_s}{3}$， $m_s$ 是弹簧的实际质量。

本实验通过改变 $m_1$ 测出相应的 T，以考察 T 和 m 的关系，从而求出 $m_0$ 和 k。

## 四、实验内容

### 1. 气垫导轨水平的调节

可以用两种方法对导轨的水平进行调节。

方法一：使用开孔挡光片，智能测时器选在 2pr 功能档。让光电门 A、B 相距约 60 cm（取导轨中央位置），给滑块以一定的初速度（Δt₁ 和 Δt₂ 控制在 20～30 ms 内），让它在导轨上依次通过两个光电门。若滑块在同一方向上运动的 Δt₁ 和 Δt₂ 的相对误差小于 3%，则认为导轨已调到水平，否则重新调整水平调节旋钮。

方法二：使用不开孔挡光片，智能测时器仍然选在 2pr 功能档。让光电门 A、B 相距 40 cm 左右，给滑块以一定初速度，比较滑块第一次滑过两个光电门的时间间隔和反弹回来第二次经过两个光电门的时间间隔的大小，如果之间误差小于 5%，则认为导轨已经调到水平，否则应重新调整水平调节旋钮。

### 2. 研究弹簧振子的振动周期与振幅的关系

为准确测定弹簧振子的周期，可用光电门和智能测时器。将光电门（当只用一个时，另一个也必须与测时器相连）置于滑块的平衡位置功能点。

先将测时器设置于 6pd（测周期）功能档。

按 “选择” 键当屏幕显示为 6pd 时，即为测周期功能。

根据实验内容，可预先设置周期数。一般而言，所设置的周期数不宜太多，因为导轨的摩擦毕竟不可忽略，但也不宜太少，使误差太大。进入测试后，屏幕上显示的是剩余周期数。

周期数的预置方法是：当屏幕显示为 6pd 时，按动 “执行” 键，显示为 0。每按一次 “选择” 键，显示加 1；当达到预定值后，将滑块拉离平衡位置后再按一次 “执行” 键，即进入测周期操作，此时屏幕上显示 “yes”。

测试时，使用开孔挡光片。每挡光两次，显示的预置数就减 1。最后一次挡光后，显示出时间的总值，单位为毫秒（ms）。

接着用手将滑块拉离平衡点 6.00 厘米（即选定某一振幅），再按 “执行” 键，放手让其运动。设滑块第一次自左向右运动，当滑块第一次通过光电门时测时器开始计时；当滑块反方向再次通过光电门时预置数减 1；当滑块自左向右第三次通过光电门时预置数再减 1。由此可见，滑块作一次完整振动，预置数减 2。依此类推，滑块作 n 次完整振动，预置数减 2n。

因此，要测滑块振动 n 个周期的时间，预置数应设定为 2n。当最后一次挡光预置数减完后，显示屏上出现总时间 t，由此可得周期 $T = \frac{t}{n}$。

将滑块拉离平衡位置 8.00、10.00、12.00（厘米），重新测量周期，每改变振幅一次，需测量周期 6 次，并测量滑块和弹簧的质量，根据测量结果探讨周期与振幅是否有关，并利用式（3-4-7）计算不同振幅下弹簧的倔强系数。


### 3. 观测简谐振动周期 T 与 m 的关系，并求出弹簧的倔强系数 k 与有效质量 $m_0$

在滑块上安装重块（矩形金属片）以改变滑块质量 $m_1$，将滑块拉离平衡位置 6.00 厘米时，测出相应的周期 T，根据式（3-4-7），得

$$T^2= \frac{4\pi^2}{k}m_1 + \frac{4\pi^2}{k}m_0 。$$

上式表明，当弹簧倔强系数 k 一定时， $T^2$ 和 $m_1$ 成线性关系。 $T^2 - m_1$ 图线为一条直线，其斜率为 $\frac{4\pi^2}{k}$，其截距为 $\frac{4\pi^2}{k}m_0$。

取不同的 $m_1$ 值 4 次：①滑块本身；②滑块加两个螺丝；③再加两个骑码；④最后再加两个骑码（共 4 个）。分别测出相应的 T 并验证式（3-4-8）。注意称取质量时需倒着称回去，即先称取滑块 + 两个螺丝 + 四个骑码的质量。
## 五、注意事项

1. 由于弹簧的弹性限度很小，绝不能用手随便拉伸弹簧，否则弹簧超过弹性限度，就不能恢复原状。
2. 注意实验数据有效数字应保留的位数（其中原始数据按仪器显示来读取；非原始数据应按有效数字的运算法则来进行保留）。
