### 这是一部分大物基础实验实验报告的电子版本，你可以使用写字机之类的设备打印出来节省时间精力。
### markdown文件方便编辑，留出的空位是给公式或者图表之类的，导出的docx文件可以被写字机直接读取；attachments目录比较杂乱，有实验需要的表格（通常是latex），数据处理部分的作图或者现成的图片（通常是python或者excel），以及可以参考的实验数据和结果。
#### 为了编译或者运行它们，你需要：
#### ​	1.代码编辑器，用来编辑markdown，latex和python文件。如果使用vscode：
##### ​	安装vscode
```powershell
winget install Microsoft.VisualStudioCode
```
##### ​	vscode中安装以下扩展

> [!NOTE] Latex
> **Latex Workshop**;
> Latex Viewer;
> Latex Utilities

> [!NOTE] Markdown
> **Markdown All in One**;
> **Markdown Preview Enhanced**;
> Markdown Preview Github Styling
> 对于Markdown，个人更推荐使用**Obsidian**

> [!NOTE] Python
> **Python**;
> **Python Debugger**;
> Python Extension Pack;
> **Pylance**;
> 以及运行中所需要的**PyPI**库

需要Pandoc来导出md为docx文件，如果使用obsidian，最好配合Enhancing Export插件，以及，最好需要一个ai工具来快速完成latex繁复的语法。

#### ​	2.可以解释或者编译这些脚本的环境，主要是python解释器和texlive：
##### ​	​安装python解释器
```powershell
winget install Python.python.3.14
```
##### 如果你想为需要的pip库建立一个独立的环境防止冲突
```powershell
winget install uv
uv venv
uv pip install ...
```
##### ​	安装texlive：

> [Windows - TeX Live - TeX Users Group](https://tug.org/texlive/windows.html)

自动安装完成后添加环境变量到Path
注意编译器不支持中文路径，请不要在原路径下直接编译tex文件

> ../texlive/2025/bin/windows
